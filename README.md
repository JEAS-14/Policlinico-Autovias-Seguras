# Policlinico-Autovias-Seguras
Sitio web corporativo y sistema de citas para Policlínico Autovías Seguras. Desarrollado por Juan Arango.

Herramientas
- Java Spring Boot con templates (Thymeleaf)
- Arquitectura MTC (Model-Template-Controller) en lugar de MVC
- Landing page con un formulario
- Enfoque simple sin frontend separado

Lenguajes
- java version "21.0.7" 
- Apache Maven 3.9.9 


Comandos
- inicializar - mvn spring-boot:run
- reload mvn -  mvn clean compile
